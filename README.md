# User Dashboard Application

## Overview

The User Dashboard Application is a full-stack web application that allows users to view, manage, and update user information through an interactive dashboard. The application demonstrates modern web development practices using Angular for the frontend and a RESTful API backend.

## Features

* User registration and login
* View a list of users
* Add new users
* Edit existing user details
* Delete users
* Search and filter users
* Responsive dashboard interface
* Real-time data updates through API integration

## Technology Stack - 

### Frontend

* Angular
* TypeScript
* HTML5
* CSS3
* RxJS
* Angular Services

### Backend

* Node.js
* Express.js
* REST APIs

### Database

* MongoDB

## Project Architecture

The application follows a client-server architecture:

* Angular frontend handles the user interface and user interactions.
* Express.js backend exposes REST APIs for CRUD operations.
* MongoDB stores user data.
* Angular services communicate with backend APIs using HttpClient.

## Key Functionalities

### User Management

Users can perform complete CRUD operations:

* Create new user records
* Read and display user information
* Update existing user details
* Delete user records

### State Management

The application uses RxJS BehaviorSubject to maintain and update user data across components efficiently.

### API Integration

Frontend components communicate with backend APIs through Angular services, ensuring clean separation of concerns and maintainable code.
